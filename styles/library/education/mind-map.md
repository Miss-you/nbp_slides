# 思维导图风格

## 风格ID
mind-map

## 风格名称
思维导图有机网络风格PPT

## 适配模型
- Nano Banana Pro (gemini-3-pro-image-preview)
- NotebookLM
- Youmind
- Listenhub
- Lovart

## 原始提示词
```text
Neural network, brainstorming web. Layout: Central node with organic branches extending outward. Visuals: Nodes are colored bubbles connected by curved bezier lines. Style: Organic, biological interface, clean UI. White background with colorful distinct branches.
```

## 风格描述
思维导图有机网络风格将神经网络的生物学美感与现代清洁UI界面设计融合，创造出一种同时兼具科学感和视觉美感的信息呈现方式。页面以一个中央核心节点（大型彩色气泡）为起点，有机分支从中心向四周自然延伸，每条分支使用独立的主题色彩，互不干扰。节点呈现为大小不等的彩色圆形气泡，通过平滑的贝塞尔曲线连接，整体形态类似神经突触网络或生命树的生长形态，而非机械化的直线流程图。白色背景保持界面的干净感，让丰富的彩色分支自然突出。这一风格适合概念拆解、知识关联展示、系统架构说明、课程结构总览及任何需要呈现"中心到边缘"层级关系的场景。

## 基础提示词模板

你是一位信息架构与视觉设计专家，请生成16比9的思维导图有机网络风格演示文稿幻灯片。

**视觉风格与美术指导**

整体风格定位：有机生物界面（Organic Biological Interface）与清洁UI设计（Clean UI Design）的融合。画面应同时具有神经网络的有机流动感和现代数字界面的精准简洁感。严禁使用直角连线或流程图式矩形框，所有连线必须是平滑曲线，所有节点必须是圆形或椭圆形气泡。

中央节点：页面视觉中心放置一个大型圆形气泡节点，直径约为页面宽度的15-18%。颜色使用饱和度中等的深色圆形（如深紫 #5C2D91、深蓝 #0066CC 或深青 #007A7A），内置白色粗体主题文字，字号突出，代表整张思维导图的核心概念。

分支结构：从中央节点向外辐射3-6条主分支，每条主分支使用独立的鲜明主题色（色相间隔60度以上，确保视觉区分）。每条主分支延伸出2-4条次级分支。分支的有机程度体现在：线条应顺势弯曲延伸，方向自然（类似树木枝干的生长趋势），线条粗细随层级递减（主分支粗，次级分支细，三级分支最细）。

节点气泡规格：
- 一级子节点（主分支末端）：中型气泡，直径约8-10%页面宽度，使用各自主题色的较深色调填充，白色文字；
- 二级子节点：小型气泡，直径约5-6%页面宽度，使用各自主题色的较浅色调填充（透明度80%），深色文字；
- 三级子节点（如有）：最小气泡，圆形或胶囊形，使用极浅色填充，深色文字；
- 所有气泡均带有轻微阴影（Box Shadow），增加层次浮动感。

贝塞尔连线：节点之间的连线必须使用平滑的贝塞尔曲线（Bezier Curve），曲线弯曲方向顺应节点相对位置，形成自然流动的弧度。线条粗细：主分支3-4px，次级分支2px，三级分支1-1.5px。线条颜色与目标节点所属分支主题色一致，透明度85%。线条末端可添加小圆点装饰（代替箭头）。

**配色方案**

背景色：纯白 #FFFFFF
中央核心节点：深紫 #5C2D91 或深蓝 #1A56DB
分支一主题色：珊瑚红 #FF4757
分支二主题色：天空蓝 #1E90FF
分支三主题色：薄荷绿 #2ED573
分支四主题色：温暖橙 #FFA502
分支五主题色：薰衣草紫 #A29BFE
分支六主题色：金黄色 #ECCC68
文字色（深色节点上）：纯白 #FFFFFF
文字色（浅色节点上）：深炭灰 #2F3542
辅助阴影：rgba(0,0,0,0.1)

**字体排版**

中央节点文字：粗体无衬线体，字号最大，白色，居中对齐。
一级子节点文字：半粗体无衬线体，字号次之，白色，居中对齐。
二级/三级节点文字：常规无衬线体，字号较小，深色或白色（视背景而定），居中对齐。
所有字体保持统一的几何感无衬线字族，确保界面的清洁感。

## 页面类型模板

### 封面页模板
构图逻辑：白色背景，页面正中央放置大型深色核心气泡，内置主题名称文字。从核心节点向外辐射出完整的思维导图网络，分支延伸至页面边缘区域（保留适当留白）。每条主分支使用独立的鲜明颜色，次级节点颜色渐浅。整体画面呈现向四周均匀辐射的有机网状形态。页面左下角或右下角添加极小字号的副标题文字（演示者名称、日期等），不干扰主体思维导图视觉。

使用场景：PPT第一页，知识体系总览、概念系统介绍、课程结构展示。

### 内容页模板
构图逻辑：白色背景，页面展示思维导图的某一局部放大视图——中央节点代表当前主题，2-4条主分支向外延伸，每条分支展开2-3层子节点。重点突出当前讲解的分支（颜色饱和度更高，气泡尺寸稍大），其余分支用降低透明度的方式保留但弱化。分支内容以简洁短语为主（每个节点不超过6个汉字），配合小型图标（可选）。

使用场景：逐步讲解概念的某一分支、知识点拆解、系统某模块详述。

### 数据页模板
构图逻辑：将数据信息整合进思维导图节点中——中央节点展示总量或总体结论，各主分支节点展示分类数据，节点气泡尺寸可根据数据量大小动态调整（气泡越大=数量越多），体现数据的相对比例关系。连线粗细也可根据数据权重调整。分支颜色继承标准配色方案，确保视觉一致性。关键数字使用白色粗体在气泡中心突出显示。

使用场景：展示分类统计数据、市场份额分布、概念权重比较等。

```
{基础提示词模板}

请生成封面页。白色背景，页面正中央放置深紫色 #5C2D91 大型核心气泡，内置白色粗体文字：
[核心主题名称]

从核心节点向外辐射[N]条主分支，分支颜色依次为珊瑚红 #FF4757、天空蓝 #1E90FF、薄荷绿 #2ED573、温暖橙 #FFA502（按需选择）。

每条主分支延伸出2-3个次级节点，内容为：
[分支内容列表]

所有连线使用平滑贝塞尔曲线，白色背景，充分留白。
```

## 技术参数

### Nano Banana Pro配置
- 模型：gemini-3-pro-image-preview
- 比例：16:9
- 分辨率：2K (2752x1536) 或 4K (5504x3072)
- 响应模式：IMAGE

### 建议设置
- 推荐分辨率：2K（平衡质量和生成速度）
- 适用于：知识体系总览、课程结构展示、概念关联分析、产品功能架构、系统设计说明、头脑风暴结果可视化等场景
- 风格特点：有机流动、层级清晰、色彩丰富、神经网络感
