```text
Step-by-step, instruction manual (Ikea style). Style: Flat 2.0 vector illustration. Layout: S-Curve path winding down the page. Visuals: 5 distinct steps represented by character illustrations interacting with objects. Connectors: Dotted lines. Colors: Bright primary colors on white.
```

## 风格描述
宜家说明书风格以全球最知名的产品装配手册为灵感，将复杂的分步流程以极度清晰、无需语言辅助的视觉方式呈现。核心特征是扁平化2.0矢量插画风格：画面干净、线条精准、角色动作明确。独特的S形曲线路径将各步骤从页面顶部蜿蜒引导至底部，每个步骤由独立的角色插画（人物与物品交互）组成，步骤之间以虚线连接符串联，形成流畅的视觉叙事。配色严格使用白色背景叠加明亮原色（红、蓝、黄），确保信息在全文化背景下都能被直觉性理解。这一风格尤其适合操作培训、产品使用指南、标准化流程展示及跨国多语言场景。

## 基础提示词模板

你是一位专业的信息设计师，请生成16比9的宜家说明书风格演示文稿幻灯片。

**视觉风格与美术指导**

插画风格：扁平化2.0矢量插画（Flat 2.0 Vector Illustration）。线条干净、精确，形状简洁。人物角色使用极简几何化表达——无面部细节或仅有极简表情，肢体动作清晰、指向明确，强调动作本身而非个人特征。所有插画应达到"无需文字即可理解步骤"的清晰程度。

布局路径：采用S形曲线路径（S-Curve Path）贯穿整个页面，将步骤1至步骤5（或更多）从页面左上方引导至右上方，再折返至左中，再延伸至右下，最终形成连贯的蜿蜒流向。路径本身作为视觉引导线，用虚线（Dotted Line）绘制，虚线圆点间距均匀，颜色为浅灰 #CCCCCC 或步骤对应的主题色调。

步骤区块：每个步骤由以下三部分组成：（1）步骤编号——圆形或方形底色数字标签；（2）角色插画——表现人物正在执行该步骤的动作，与相关物品产生明确交互；（3）简短标注——步骤说明文字，字数极少，以名词或动词短语为主，类似宜家说明书中的零件清单标注。

连接符：步骤之间统一使用虚线（Dotted Line）连接，虚线末端可添加小型箭头指示方向，禁止使用实线或复杂曲线连接。

**配色方案**

主背景色：纯白 #FFFFFF
原色红（强调/步骤标签）：#FF3B30
原色蓝（信息/装饰）：#007AFF
原色黄（高亮/注意）：#FFD600
深炭灰（线稿/文字）：#1A1A1A
浅灰（次要元素/虚线）：#CCCCCC
黑色（轮廓描边）：#000000

**字体排版**

步骤编号：粗体无衬线体，置于彩色圆形或方形标签内，白色数字。
步骤说明：极简无衬线体，字号小，字符数量严格控制，推荐每步不超过8个汉字或12个英文字符。
页面标题（如有）：黑色粗体无衬线体，左对齐，简洁明了。
禁止使用：装饰性字体、斜体、超过两种字重混用。

## 页面类型模板

### 封面页模板
构图逻辑：白色背景，页面中央绘制一幅概括整体流程的大型插画，展示角色与最终成品/目标的交互场景。左上角放置产品/课程名称（黑色粗体），右上角可添加步骤总数标注（如"5 STEPS"用原色红标签展示）。底部绘制一排小型步骤预览图标，以虚线串联，作为目录预告。整体干净、留白充足。

使用场景：PPT第一页，展示流程总览、课程介绍、操作手册封面。

### 内容页模板
构图逻辑：页面整体展示S形曲线路径，将5个步骤区块从左上蜿蜒至右下。每个步骤区块包含编号标签（原色轮换）、角色与物品交互插画、极简步骤说明文字。步骤之间用浅灰虚线连接，末端带箭头。页面留白充足，背景保持纯白，避免任何装饰性背景元素干扰视觉流。

使用场景：展示操作步骤、安装流程、使用方法、标准化作业程序（SOP）等。

### 数据页模板
构图逻辑：将数据转化为"量化说明书"形式——使用角色插画配合数字标注呈现统计信息（例如：角色手持物品，旁边标注数量或百分比）。使用原色系简洁柱状图或图标矩阵（Icon Matrix）补充量化数据，图表线条风格与说明书插画保持一致，干净精确。关键数字使用原色红或原色蓝的圆形标签放大展示。

使用场景：展示数量统计、完成率、步骤耗时对比、物料清单数据等。

- 比例：16:9
- 分辨率：2K (2752x1536) 或 4K (5504x3072)

- 推荐分辨率：2K（平衡质量和生成速度）
