# 故事板旅程风格

## 风格ID
storyboard

## 风格名称
故事板叙事旅程风格PPT

## 适配模型

## 原始提示词
```text
User experience, comic strip, narrative. Visuals: Consistent character moving through a scenario. Text: Captions under each frame. Style: Vector art, semi-realistic.
```

## 风格描述
故事板叙事旅程风格借鉴电影分镜表与漫画条格的经典叙事结构，将信息转化为一段连贯的视觉故事。页面被划分为若干等分画格（Frame），一个造型固定、外观一致的矢量角色贯穿始终，在每个画格中经历场景变化、情绪演进或动作推进，配合画格底部的简短说明文字（Caption），共同构成一段完整可读的视觉叙事。风格定位为矢量艺术与半写实（Vector Art, Semi-Realistic）的融合：角色造型简洁有辨识度，面部情绪可读，环境背景以简洁矢量块面呈现。这一风格特别适合用户体验（UX）流程演示、产品使用场景说明、服务蓝图展示、客户旅程地图（Customer Journey Map）及任何需要以"角色视角"讲述流程的场景。

## 基础提示词模板

你是一位专业的故事板艺术家和UX设计师，请生成16比9的故事板叙事旅程风格演示文稿幻灯片。

**视觉风格与美术指导**

整体风格定位：矢量艺术（Vector Art）与半写实（Semi-Realistic）融合风格。角色造型具有一定的立体感和情绪表达力，但整体保持矢量简洁的轮廓，不追求照片级写实。色彩使用平涂为主，允许简单的阴影（最多两层明暗），禁止使用复杂渐变或材质贴图。

画格布局（Frame Layout）：页面按照内容帧数均匀划分画格，每个画格之间有细边框（1-2px，颜色 #333333）分隔，类似漫画条格或电影分镜格式。标准布局方案：
- 3格横排（推荐简洁流程）
- 4格横排（推荐标准叙事）
- 2×2格（推荐四象限场景）
- 6格（3×2网格，推荐复杂旅程）
每个画格比例接近4:3或3:2（根据总格数自适应）。

角色一致性（Character Consistency）：整张幻灯片中的主角角色必须保持视觉一致性——相同的发型、服装配色、体型比例。角色风格为简洁矢量半写实风：头身比约1:3到1:4，面部有清晰的情绪表情（微笑、思考、惊讶、满意等），服装颜色作为角色识别标志（如始终穿蓝色上衣）。不同画格中角色的动作和情绪随场景推进而变化。

场景背景（Scene Background）：每个画格的背景以简洁的矢量色块和线条搭建环境（办公室、手机界面、户外等），背景细节适度，以不抢夺角色视觉焦点为原则。背景色调可在不同画格间有所变化（反映情境变化），但整体配色保持协调。

说明文字（Caption）：每个画格底部留出固定高度的说明区域（约画格高度的20%），使用浅灰或白色底色与画面区域分隔，内置简短的场景说明文字（不超过20个汉字），字体为清晰可读的无衬线体，字号适中。

画格内文字气泡（Speech/Thought Bubble）：如需表现对话或内心想法，使用经典漫画气泡——对话用圆角矩形气泡配尾部指向角色嘴部，想法用圆形泡泡串联的气泡指向角色头部。气泡背景白色，边框颜色与角色主题色协调。

**配色方案**

页面背景（外框区域）：浅灰 #F0F0F0 或极浅米色 #FAF8F5
画格内边框：深炭灰 #333333
说明文字区域底色：白色 #FFFFFF 或浅灰 #F8F8F8
角色主色（服装标识色）：天空蓝 #3A86FF
角色辅色（皮肤/头发）：暖米肤色 #FDBCB4，深棕发色 #5C3D2E
环境背景色一（室内/暖）：浅橙米 #FFF3E0
环境背景色二（数字界面）：浅蓝白 #E8F4FD
环境背景色三（户外/活动）：浅绿白 #F1F8E9
强调色（高亮元素）：亮橙 #FF6B35
情绪正面色（成功/满意）：薄荷绿 #A8E6CF
情绪负面色（困境/问题）：浅红橙 #FFCCBC
Caption文字色：深炭灰 #424242

**字体排版**

画格说明（Caption）：无衬线体，常规字重，字号14-16pt，深炭灰 #424242，居中对齐。
气泡对话文字：无衬线体，常规字重，字号较小，深炭灰，居中对齐。
页面标题（如有）：粗体无衬线体，字号18-20pt，左上角或居中，深炭灰或角色主色。

## 页面类型模板

### 封面页模板
构图逻辑：页面采用"电影海报"式构图——左侧或中央放置完整角色站姿插画（展示主角全身），角色背后的背景以简洁矢量场景元素构成（代表整个旅程的起点或终点环境）。右侧或下方放置标题文字和副标题。页面底部可绘制一排小型分镜缩略图（不含文字），作为整体故事预告，以细边框分隔，配箭头串联，暗示后续故事推进。整体保持矢量艺术的干净感，色调温暖积极。

使用场景：PPT第一页，UX演示开场、产品故事介绍、客户旅程开始。

### 内容页模板
构图逻辑：页面均匀划分为3-4个横向画格，主角在每个画格中推进场景。每格内：背景反映当前环境状态，角色动作和表情对应当前步骤的情绪/行为，可选的对话/思维气泡表达内心想法，底部说明区域文字说明该步骤。画格间用细边框清晰分隔。整体色调根据旅程阶段（挣扎→转折→成功）有层次地从冷暖色过渡。

使用场景：展示用户旅程各阶段、产品使用流程、服务接触点场景、操作步骤叙事。

### 数据页模板
构图逻辑：将数据融入故事格——主角在一个画格中与数据可视化元素（如屏幕上的图表、手持的统计卡片）互动，角色表情和姿态与数据结论情绪匹配（数据正向时角色喜悦，数据显示问题时角色困惑）。画格旁侧或下方补充简洁的矢量图表（保持矢量半写实风格，圆角样式），说明区域文字标注关键数据。整体用视觉叙事赋予数据情感温度。

使用场景：用叙事化方式呈现用户研究数据、NPS分数、转化率、调研发现等。

## 技术参数

### Nano Banana Pro配置
- 模型：gemini-3-pro-image-preview
- 比例：16:9
- 分辨率：2K (2752x1536) 或 4K (5504x3072)
- 响应模式：IMAGE

### 建议设置
- 推荐分辨率：2K（平衡质量和生成速度）
- 适用于：用户体验（UX）流程演示、产品使用场景说明、客户旅程地图、服务蓝图、创业Pitch故事化叙述、品牌故事、培训流程场景化等场景
- 风格特点：叙事连贯、角色一致、情绪可读、漫画分镜感
