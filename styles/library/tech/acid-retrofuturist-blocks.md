# 酸性色块复古未来风

## 风格ID
acid-retrofuturist-blocks

## 风格名称
酸性色块复古未来风

## 适配模型
- Nano Banana Pro (gemini-3-pro-image-preview)

## 风格来源
DrawPPT 风格提取，来源候选 `acid-retrofuturist-color-block`。原候选来自高密度技术海报描述，已改写为通用 16:9 PPT 风格，并弱化单张竖版海报语境。

## 风格描述
酸性色块复古未来风把 Y2K tech nostalgia、早期互联网硬件、CD 光盘、像素化设备、贴纸标签和高对比酸性色块组合成模块化知识海报。它不是夜店霓虹，也不是纯 8 位游戏截图，而是 1999 至 2005 年技术消费品广告、开发者杂志和地下电子海报混合后的复古未来视觉。

该风格适合技术趋势、AI 产品、开发者工具、工具链速查、复杂概念拆解和面向年轻技术受众的演示。页面以深炭黑颗粒背景为底，叠加赛博黄、电橙、霓虹绿、酸性粉和冷银灰模块，让知识像一张带技术规格、警告贴纸和硬件切片的未来怀旧海报。

## 基础提示词模板

你是一位专业的复古未来技术海报与演示设计师，请生成适配 16:9 演示文稿的酸性色块复古未来风幻灯片。

全局视觉语言：酸性色块复古未来风。画面像一张 Y2K 时代的技术知识海报，深色颗粒底上堆叠高饱和 acid color 色块、贴纸标签、硬件碎片、技术规格框和模块化信息网格。

背景与底色：
- 背景使用深炭黑 `#111111` 或墨黑 `#050505`，叠加重颗粒噪点、轻微扫描纹、复印机粗糙纹理和低透明度像素网格。
- 可加入微弱彩虹光盘反射、旧显示器扫描线、破碎窗口边框和硬件线路纹理，但必须保持信息可读。
- 背景不得使用砖墙、夜生活烟雾、真实霓虹灯管或游戏关卡地貌，避免与霓虹夜生活和 8 位复古游戏风格重叠。

配色方案：
- 深炭黑 `#111111`：主背景和暗部容器。
- 复印纸白 `#F4F1E8`：正文反白、贴纸底和信息标签。
- 赛博黄 `#DFFF00`：主强调、关键结论、标题冲击色。
- 电橙 `#FF5A1F`：行动项、警告条、热区标记。
- 酸性绿 `#39FF14`：状态、增长、成功指标和线路节点。
- 激光粉 `#FF2DAA`：次强调、反差标签和视觉锚点。
- 钴蓝紫 `#4B3DFF`：冷色对比块、链接、技术背景面。
- 冷银灰 `#B8B8C8`：硬件材质、边框、规格文本和辅助线。

排版规范：
- 标题使用粗重无衬线或扩展体，字形紧凑有冲击力，可采用全大写英文编号与中文粗标题混排。
- 副标题和规格文字使用等宽字体，形成 `SPEC / INPUT / OUTPUT / PATCH / BUILD` 一类技术标签。
- 正文必须短句化，放入模块化色块、贴纸、规格卡或黑底反白框中。
- 信息标签可使用圆角小贴纸、切角矩形、条形码标签、警告条和硬件铭牌，但整体保持模块化知识海报逻辑。
- 禁止使用霓虹草书、游戏像素字体作为主体排版；允许少量像素字仅作为装饰性状态码。

材质与质感：
- 以 2D 平面海报为主，混合少量复古科技物件：低多边形旧电脑、CRT 屏幕、CD 光盘、软盘、透明塑料外壳、线缆、芯片和贴纸。
- 色块边缘可有轻微错位印刷、半调网点、粗颗粒、复印阴影和贴纸翘边。
- 硬件物件应作为视觉锚点或模块说明，不要发展成完整游戏场景或写实产品摄影。
- 禁止玻璃拟态、柔和渐变 SaaS 卡片、砖墙霓虹灯管、纯 8-bit 方块像素画面。

构图与密度：
- 16:9 页面采用不对称模块网格，建议 5 至 7 个知识模块，信息密度高但每个模块边界清楚。
- 用大色块建立主次：一个主色块承载核心结论，多个小色块承载定义、步骤、参数、风险和速查项。
- 画面可有倾斜贴纸、切角标题条、硬件碎片和规格表，但所有元素必须服务于阅读路径。
- 视觉与文字比例建议为 50:50；封面偏视觉冲击，内容页和数据页偏规格化模块。

风格边界（Do / Don't）：
- Do: 强调 Y2K 技术怀旧、酸性色块、重颗粒、贴纸标签、旧硬件、技术规格块和模块化知识海报。
- Do: 使用高对比色块表达信息层级，并用规格框、警告条和 cheat sheet 组织知识。
- Don't: 做成夜店霓虹招牌、砖墙灯管、烟雾 noir 场景或手写霓虹字体。
- Don't: 做成纯 8-bit 游戏 HUD、平台关卡、像素角色冒险或受限 FC 调色板。

渲染质量要求：生成高冲击、可读性强的 16:9 复古未来技术海报幻灯片，颗粒、套印和贴纸质感明确，色块干净有力，文字模块边界清晰。

## 页面类型模板

### 封面页模板
构图逻辑：深炭黑颗粒背景上放置一个巨大的酸性色块标题区，标题使用粗重无衬线字体。右侧或下方加入旧电脑、CD 光盘、芯片或透明塑料硬件的拼贴视觉锚点，周围散布 `SPEC-00`、`BOOT SEQUENCE`、条码、警告贴纸和小型技术参数。

使用场景：技术趋势、AI 产品、开发者分享、工具链课程和复古未来主题报告的开场页。

### 内容页模板
构图逻辑：使用不对称模块网格组织 5 至 7 个信息块。一个大色块承载主结论，其他色块分别承载定义、步骤、案例、风险和行动项。模块顶部使用等宽规格标签，色块之间用细线、箭头、贴纸编号或硬件线路连接。

使用场景：展示核心观点、技术概念拆解、工具链流程、功能清单、知识速查和方法框架。

### 数据页模板
构图逻辑：左侧放置超大数字和结论贴纸，右侧用黑底规格卡展示柱状图、矩阵、雷达图或参数表。图表使用赛博黄、酸性绿、电橙和钴蓝紫分组，坐标轴和注释使用冷银灰等宽字体。异常值用警告条或激光粉贴纸标出。

使用场景：展示指标对比、技术选型、产品参数、趋势数据、模型效果和 benchmark 结果。
