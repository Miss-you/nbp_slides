## 风格描述
渐变英雄漏斗营销风格是一套专为营销演示、用户转化流程说明、增长黑客报告和品牌战略汇报设计的现代渐变视觉主题。本风格的视觉核心是"英雄漏斗"——一个从宽到窄、3D感强烈的漏斗形状悬浮于画面中央，象征用户从认知到转化的全流程，是市场营销"转化漏斗"（Conversion Funnel）概念的视觉化具象。

配色灵感直接来源于Instagram的品牌渐变设计语言：将紫色（#833AB4）、玫瑰红（#E1306C）、橙红（#FD1D1D）和琥珀橙（#FCAF45）以平滑过渡的网格渐变（Mesh Gradient）形式融合，创造出如同活物般流动的色彩空间。这种渐变配色已成为2020年代现代品牌设计的标志性美学——充满活力、充满温度、充满人情味，同时保持足够的科技感和未来感。

网格渐变（Mesh Gradient）是本风格的技术核心：不同于简单的线性渐变或径向渐变，网格渐变通过在画面不同位置设置多个颜色控制点，让颜色以有机、非线性的方式流动和混合，产生类似极光、液态金属或油膜的视觉效果。这种渐变具有高度的独特性，每次生成的渐变图案都略有不同，天然抵制复制。

漏斗形状的3D化设计进一步强化了营销流程的视觉叙事：漏斗顶部宽大，代表广泛的潜在用户群（Awareness 认知阶段）；中部收窄，代表兴趣筛选（Interest/Consideration 考量阶段）；底部最窄，代表最终转化（Conversion 转化阶段）。漏斗表面反射渐变背景的光色，产生玻璃或金属质感的光泽，"悬浮"在渐变场中，赋予整个画面强烈的3D空间纵深感。

漏斗各层次可以标注关键指标（如"10,000 访问"、"3,200 注册"、"800 购买"），配合大号数字和精简标签，将抽象的转化概念变成直观的数据叙事。背景渐变也可以用于非漏斗页面，作为独立的视觉背景配合各种内容模块，保持风格一致性。

## 基础提示词模板

你是一位专业的营销视觉设计师，请生成现代渐变风格的16比9营销演示幻灯片，整体氛围充满活力、流动感强、现代感十足，适合展示转化漏斗、用户增长路径和营销战略。

全局视觉核心：以流动的现代网格渐变（Mesh Gradient）为视觉基底，颜色灵感来源于Instagram品牌渐变美学。渐变颜色在画面中以有机、非线性的方式流动混合，产生类似极光或液态色彩的视觉效果。

渐变配色方案：
核心渐变色板（Instagram渐变风格）：
- 紫罗兰：#833AB4
- 玫瑰红：#E1306C
- 橙红：#FD1D1D
- 琥珀橙：#FCAF45
- 蜜桃粉：#F77737

扩展渐变色板（可选替换，根据品牌调性选择）：

方案A（Instagram经典）：#833AB4 → #E1306C → #FD1D1D → #FCAF45
方案B（紫蓝极光）：#667EEA → #764BA2 → #F64F59 → #C471ED
方案C（珊瑚日落）：#FF6B6B → #FFA07A → #FFD93D → #6BCB77
方案D（青蓝海洋）：#4FACFE → #00F2FE → #43E97B → #38F9D7

背景设计：全页使用上述渐变色板生成的网格渐变作为背景。渐变应呈现以下特征：颜色在画面不同象限流动，无明确的线性方向感；颜色过渡极为平滑（无明显色带），如同柔焦摄影；整体亮度适中，确保白色文字在其上可读；可在渐变背景上叠加极细微的噪点纹理（5%以内），增加质感。

漏斗形状设计（核心视觉元素）：绘制一个从上宽下窄的3D漏斗形状（等轴测视角或45度侧视），漏斗悬浮于渐变背景中央，带有以下质感特征：
- 表面材质：半透明玻璃或磨砂亚克力，反射背景渐变的光色
- 漏斗分为3-5层（可明确分层分色），每层颜色略有差异，从顶部到底部颜色逐渐加深或加浓
- 漏斗边缘具有高光线（1-2px的白色或明亮渐变线），强调3D立体感
- 漏斗底部有轻微的投影（Soft Drop Shadow），使其悬浮感更真实
- 漏斗整体倾斜角度建议为：垂直或微微前倾（5-10度），使顶部开口面向观看者

白色文字规范：所有文字使用白色（#FFFFFF）或极浅色（#F8F8F8），确保与渐变背景形成足够对比度。标题字体使用现代无衬线（如Inter、SF Pro Display、Poppins），字重600-700。关键数字使用特大号（60-80px），说明文字使用常规号（14-18px）。可在文字背后添加半透明的白色磨砂玻璃卡片（白色10-15%透明度，backdrop-filter blur效果）以增强可读性。

装饰元素：允许在漏斗周围添加以下装饰性元素：1）流动的弧线或曲线（白色，2-3%透明度）暗示流量流向；2）小型的圆形或点阵（白色，5%透明度）作为背景装饰；3）简洁的箭头（白色半透明）指向漏斗各层，标注流程阶段。

渲染质量要求：顶级营销设计品质，如同Figma Community精选模板，色彩饱满且过渡丝滑，UI感强，适合投放于演示文稿、社交媒体和营销物料。

## 页面类型模板

### 封面页模板
构图逻辑：全页铺满方案A的Instagram风格网格渐变背景（#833AB4 → #E1306C → #FD1D1D → #FCAF45）。画面中央偏右放置一个大型3D半透明漏斗（约占页面高度70%），漏斗分3层，从上到下颜色由浅紫向深橙过渡，边缘有白色高光线。页面左侧（约40%宽度）为文字区：顶部小型白色分类标签（如"MARKETING STRATEGY 营销战略"，半透明白色胶囊背景）；中部大号白色粗体主标题（48-64px，2行，字重700）；下方副标题（白色，18px，常规字重，80%透明度）；底部演讲者信息（白色小字，60%透明度）。整体画面流动感强、活力充沛。

使用场景：营销战略演示首页、增长报告开篇、SEM/SEO汇报封面、产品GTM（Go-to-Market）方案封面。

### 内容页模板
构图逻辑：背景使用渐变色板的浅色变体（降低饱和度至40%，或使用纯白背景配彩色渐变装饰条）。页面顶部为粗体标题（渐变色文字，取主渐变两端颜色形成文字渐变效果，或白底配深色字）。主体区域使用白色半透明磨砂玻璃卡片（3-4张，均等或略有大小差异的圆角矩形）排列内容，每张卡片内有渐变色图标（取自主渐变配色）、标题和说明文字。卡片背景为白色15%透明度，边框为白色20%透明度1px，backdrop-filter blur使卡片具有玻璃质感。卡片间距均匀，整体排列在内容区横向居中。

使用场景：展示营销策略要点、用户旅程阶段、增长黑客方法论、渠道分析等。

### 数据页模板
构图逻辑：以转化漏斗数据可视化为核心。页面中央偏右放置垂直的3D分层漏斗，漏斗共分3-5层，每层用不同的渐变深浅区分。每层漏斗左侧（或右侧，交替对齐）用白色文字标注阶段名称和关键指标数字（大号，粗体，与漏斗层高度对齐）。页面左上角放置标题和简短结论语（白色）。漏斗各层之间的过渡区域用细箭头或流动线条（白色半透明）表示流量流向，旁边标注转化率百分比（如"32% 转化率"）。整体构图使数据叙事清晰且视觉化。

使用场景：展示用户转化漏斗数据、各营销渠道转化率对比、销售管道（Pipeline）数据、用户留存率分析。
