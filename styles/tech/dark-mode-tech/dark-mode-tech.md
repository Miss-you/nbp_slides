# 暗黑科技

## 风格ID
dark-mode-tech

## 风格名称
暗黑科技未来UI风格

## 适配模型
- Nano Banana Pro (gemini-3-pro-image-preview)
- Seedream

## 风格描述
面向赛博朋克、加密货币和开发者文化的未来主义暗黑科技UI风格。以深黑色与木炭灰为绝对背景基底，以霓虹青色和品红色作为功能性高亮点缀。视觉核心为发光的细线电路图案、玻璃拟态卡片效果和等宽编程字体排版，整体呈现精密电路原理图的技术美学。画面散发浓郁的开发者美学：终端命令行、代码块、十六进制数据流和系统监控界面元素穿插于内容之间，在科技理性与赛博朋克美感之间寻找完美平衡点。

## 基础提示词模板

你是一位专业的未来科技UI演示设计师，请生成面向开发者和赛博朋克文化的16比9暗黑科技风格幻灯片。

全局视觉语言方面，背景必须为深邃的纯黑（#0A0A0F）或木炭黑（#12141A），代表无尽数字虚空。所有UI元素的霓虹高光严格限定在两种颜色：霓虹青色（#00F5FF）用于主要强调线、边框发光和关键数据；霓虹品红（#FF2D78）用于警告状态、对比强调和次要装饰。辅助颜色包含代码绿（#39D353，仅用于代码块）、数据蓝（#1E90FF，用于图表填充）和木炭灰（#1E2028，用于卡片背景）。所有霓虹元素必须带有相应颜色的外发光效果（blur radius 8至15px），模拟真实LED屏幕的光晕渗透。

关于画面视觉元素模块，请融合以下核心要素：

1. 电路背景纹理全画面底层铺设极细的电路板走线图案（线宽0.5px，颜色为#1A2A3A）：横竖正交线段加90度折角，节点处有微小圆点，整体呈现PCB电路板俯视图的几何美感。该纹理极度低调（透明度15%至25%），仅作为背景层次感的来源，不干扰前景信息。在特定位置（靠近霓虹元素处）电路线条发光强度升高，模拟电流流动的视觉轨迹。

2. 玻璃拟态卡片所有内容容器必须使用玻璃拟态（Glassmorphism）效果：卡片背景为半透明深灰（#1E2028，透明度75%），模糊效果（backdrop blur 12px），精致的1px边框（颜色为霓虹青#00F5FF，透明度30%至50%），顶部和左侧边缘有细微的高光线条（白色，透明度10%至15%）。卡片投影为霓虹色外发光而非传统灰色投影。

3. 等宽编程字体系统标题使用Geist Mono或JetBrains Mono等宽字体，所有字母大写，字间距宽松（letter-spacing: 0.2em）。正文同样使用等宽字体，行间距1.6，颜色为浅灰（#B0B8C8）。关键数据和代码片段使用代码绿（#39D353）或霓虹青（#00F5FF）高亮。技术类数值前缀符号（$、#、0x、>_等）以品红色（#FF2D78）显示，增强开发者美感。

4. 技术装饰元素在画面角落或边缘点缀以下元素：十六进制数据流（0x开头的随机16进制字符串，极小字号，低透明度）、扫描线（水平细线以固定间距覆盖画面，透明度3%至5%）、角落UI装饰框（L形直角线条，模拟HUD界面边框）、坐标数值标注（如[x:1920 y:1080]的系统参数显示）。

渲染质量要求：高精度UI设计风格，所有发光效果使用真实的模糊光晕而非简单描边，整体画面黑暗沉浸，霓虹高亮点睛，营造在超算数据中心深处工作的沉浸体验。

## 页面类型模板

### 封面页模板
构图逻辑：纯黑背景叠加低透明度电路纹理。中央放置1至2个玻璃拟态大卡片，标题文字以霓虹青色等宽字体大字显示（带外发光），副标题以浅灰等宽小字显示。画面中央或背景层叠加一个大型几何图形（六边形网格、圆形技术图标或数据球），以霓虹线条勾勒，带强发光效果。底部放置扫描线装饰和坐标参数文字。

使用场景：PPT的第一页，展示技术主题与暗黑科技基调。

### 内容页模板
构图逻辑：使用2至4个玻璃拟态卡片以网格布局排列，卡片间距均匀，保留足够的黑色呼吸空间。每张卡片顶部有霓虹青色细线作为顶边强调，卡片内标题用霓虹色高亮，正文用浅灰等宽字体。卡片左上角放置L形HUD装饰框，右下角放置卡片编号（如[01]、[02]）。背景电路纹理在各卡片之间的间隙处轻微发光，模拟数据流在节点间流动。

使用场景：展示核心观点、技术要点、功能列表等。

### 数据页模板
构图逻辑：左侧三分之一放置关键数据统计（超大霓虹青色数字，等宽字体，带光晕效果），每个数据下方有细小说明文字。右侧三分之二放置主图表区域：图表以玻璃拟态卡片为容器，柱状图或折线图以霓虹青色和品红色双色系绘制，图表背景有细网格参考线（透明度10%的白色横线），数据节点处有小圆形发光点。底部全宽横跨一条水平进度条或时间轴，以霓虹色标记重要节点。

使用场景：展示数据、统计信息、对比分析、系统状态监控等。
